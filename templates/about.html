{% if session.get('language') == 'mr' %} उद्दिष्ट / दृष्टी {% else %} Objective / Vision {% endif %}

{% if session.get('language') == 'mr' %} या वेब-आधारित प्रणालीचे मुख्य उद्दिष्ट विद्यार्थ्यांच्या प्रोफाइलनुसार योग्य शिष्यवृत्ती योजना शोधून देणे आहे. प्रणाली विद्यार्थ्यांच्या टक्केवारी, जात, कौटुंबिक उत्पन्न आणि वय यावर आधारित योग्य योजना फिल्टर करते. {% else %} The primary objective of this web-based application is to bridge the information gap by providing a personalized interface that identifies eligible scholarship schemes based on a student’s profile including percentage, category, income and age. {% endif %}

{% if session.get('language') == 'mr' %} परिणाम {% else %} Outcome {% endif %}

{% if session.get('language') == 'mr' %} या प्लॅटफॉर्मचे उद्दिष्ट वापरकर्त्यांसाठी शिष्यवृत्ती शोधणे सोपे करणे आहे. वापरकर्त्याच्या माहितीच्या आधारे प्रणाली योग्य योजना सुचवते. {% else %} The core philosophy of this platform is to simplify the user journey. Instead of the user searching for scholarships, the scholarships find the user based on their profile. {% endif %}

{% if session.get('language') == 'mr' %} डेटा संकलन {% else %} Data Acquisition and Aggregation {% endif %}

{% if session.get('language') == 'mr' %} शिष्यवृत्तीची माहिती MahaDBT, National Scholarship Portal आणि इतर स्रोतांमधून गोळा केली जाते आणि त्यावर आधारित प्रणाली योजना फिल्टर करते. {% else %} Scholarship data is collected from MahaDBT, the National Scholarship Portal (NSP), and other verified sources and then processed to filter schemes based on eligibility conditions. {% endif %}
